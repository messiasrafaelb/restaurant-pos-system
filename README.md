# 🍔 Luizão Lanches - PDV (Ponto de Venda)

Bem-vindo ao repositório do **Luizão Lanches**, um sistema de Ponto de Venda (PDV) completo, web e responsivo, desenvolvido para gerenciar pedidos, estoque e fluxo de caixa de forma rápida e intuitiva.

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

* **Backend:** [Node.js](https://nodejs.org/) com [Express](https://expressjs.com/)
* **Frontend:** [EJS](https://ejs.co/) (Embedded JavaScript templates), HTML5, CSS3 (responsivo) e JavaScript Vanilla
* **Banco de Dados:** [PostgreSQL](https://www.postgresql.org/)
* **Persistência/Drivers:** `pg` (PostgreSQL client para Node.js)

---

## 🛠️ Funcionalidades Sistema

* **Painel do Operador (Frente de Caixa):** Interface ágil para lançamento de pedidos de lanches, bebidas e combos.
* **Gestão de Cardápio:** Cadastro, edição e exclusão de produtos, categorias e preços.
* **Controle de Estoque:** Baixa automática de insumos e produtos conforme as vendas são realizadas.
* **Relatórios de Vendas:** Visualização do faturamento diário, produtos mais vendidos e histórico de pedidos.

---

## 📦 Como Executar o Projeto

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:
* [Node.js](https://nodejs.org/en/) (versão LTS recomendada)
* [PostgreSQL](https://www.postgresql.org/) ativo e rodando
